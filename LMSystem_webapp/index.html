About us

The "Library online" is designed with vision to digitilize small scale library, so that it will reduce burden on Librarians and will reduce their work. It is achievedby using Hyper Text markup Language (HTML), Cascading Style Sheet (CSS), Bootstrap, Javascript (Js), PHP and MySQL as database by Shashi Prakash Shah, a B.tech student as his personal project.
